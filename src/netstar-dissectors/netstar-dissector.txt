L2:
SOURCE_HARDWARE      -> hardware source
DESTINATION_HARDWARE -> hardware destination

L3:
PROTOCOL             -> ethernet type name / hexadecimal 0x0000
SOURCE_ADDRESS       -> source address
DESTINATION_ADDRESS  -> destination address

L4:
PROTOCOL             -> protocol name / hexadecimal 0x00
SOURCE_PORT          -> source port
DESTINATION_PORT     -> destination port
INFORMATION          -> application protocol fingerprint

PAYLOAD:
PAYLOAD              -> packet payload/data bytes in hexadecimal/octal/decimal/ascii format

  network_macaddr_format(&packet->layer2.smac, smac, sizeof(smac)), netstar_vendors_namebymac(&packet->layer2.smac));

  netstar_vendors_namebymac(&packet->layer2.dmac), network_macaddr_format(&packet->layer2.dmac, dmac, sizeof(dmac)));
