

# NETSTAR - Network Stealth Traffic Analyzer & Redirector

**NETSTAR** is a powerful tool designed for educational and ethical research in network security. It provides a flexible platform for analyzing, monitoring, and interacting with network traffic across Layers 2, 3, and 4 of the OSI model.

## Features

* Packet capture and injection
* Host scanning (IPv4/IPv6)
* ARP/NDP/DNS spoofing
* Port redirection
* Advanced filtering and stealth operation
